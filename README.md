# Proyecto de Automatización de Reservas con n8n

## Descripción
Este proyecto tiene como objetivo el diseño y despliegue de una infraestructura mínima viable (MVP)
para la gestión de reservas de aulas y dispositivos en un entorno educativo.

La solución se basa en una arquitectura distribuida que utiliza:
- Hosting compartido para la web
- VPS con Docker para servicios backend
- Automatización mediante n8n
- Base de datos MariaDB

## Estado del proyecto
🚧 Fase 3 – Despliegue mínimo viable en ejecución

Actualmente se encuentran operativos:
- VPS en Hostinger
- Docker y Docker Compose
- Servicio MariaDB
- Servicio n8n accesible por web
- Red Docker funcional

## Tecnologías utilizadas
- Docker / Docker Compose
- MariaDB
- n8n
- Traefik
- Ubuntu Server
- Hostinger VPS
